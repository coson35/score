# 스코어보드 컨트롤러 조작 매뉴얼

## 접속 주소
```
https://coson35.github.io/score/controller.html
```
별도 스마트폰 브라우저에서 접속합니다.

---

## 로그인

### 사용자 모드 (기본)
- 비밀번호 입력 후 "입장"
- 관리자가 미리 사용자 비밀번호를 설정해야 합니다

### 관리자 모드
- 우측 상단 "관리자" 버튼 터치 → 관리자 비밀번호 입력
- 최초 비밀번호: `admin1234` (첫 로그인 후 반드시 변경)

### 로그인 유지
- 한번 로그인하면 브라우저를 닫아도 유지됩니다
- 로그아웃 버튼을 눌러야만 로그아웃됩니다

---

## 탭 구성

하단에 5개 탭이 있습니다. (관리 탭은 관리자만 표시)

---

## ⚾ 경기 탭

### 오버레이 ON/OFF
| 버튼 | 기능 |
|------|------|
| 스코어 ON | 스코어보드 오버레이 표시 |
| 스코어 OFF | 스코어보드 오버레이 숨김 |
| 라인업 ON | 라인업 오버레이 표시 |
| 라인업 OFF | 라인업 오버레이 숨김 |

### 점수
- +1 / -1 버튼으로 원정팀, 홈팀 점수 조작
- "선후공 변경" 버튼: 홈/원정 위치 교체 (점수, 로고 포함)

### 이닝
- ◀ ▶ 버튼: 이닝 이동
- "초 ↔ 말" 버튼: 이닝 전반/후반 전환

### 카운트 (B/S/O)
- 동그라미를 터치하면 해당 수까지 채워짐
- 같은 동그라미를 다시 터치하면 하나 줄어듦
- "카운트 초기화": B/S/O 모두 0으로

### 베이스
- 다이아몬드 모양 터치: 주자 ON/OFF
- "베이스 클리어": 모든 주자 제거

## ⚙️ 경기설정 탭

### 경기 정보 입력
1. 경기명 입력 (예: "스토브리그 in 창녕")
2. 원정팀 이름 입력
   - 직접 입력하면 저장된 팀 자동완성
   - "저장된 팀" 버튼: 저장된 팀 목록에서 선택 (로고 자동 적용)
3. 홈팀 이름 입력 (동일)
4. 로고: 파일 업로드 (500KB 이하) 또는 저장된 팀에서 자동 적용

### 버튼
| 버튼 | 기능 |
|------|------|
| 경기 저장 | 프리셋으로 저장 (경기 전 미리 등록 가능) |
| 바로 적용 | 현재 경기에 즉시 반영 (점수 0으로 초기화) |

### 저장된 경기 목록
- 최신순으로 정렬
- "적용": 해당 경기 정보로 전환 (점수 초기화)
- "수정": 경기명, 팀명, 로고 수정
- "삭제": 프리셋 삭제

## 👥 라인업 탭

### 표시 모드
| 모드 | 설명 |
|------|------|
| 수비 위치 | 야구장 그림에 포지션별 선수 표시 |
| 타순 | 1~9번 타순 리스트로 표시 |

### 라인업 입력
1. 팀명 입력
2. 수비 포지션별 선수 선택 (선수 탭에서 등록한 명단)
3. 타순 1~9번 선수 선택 및 포지션 지정
4. "라인업 저장 및 표시" → 저장 + 오버레이 표시 동시 처리

### 라인업 프리셋
- 3학년/2학년 등 여러 라인업을 미리 저장 가능

#### 프리셋 사용법
1. 라인업 입력 완료
2. "라인업 프리셋 저장" 클릭 → 이름 입력 (예: "3학년 라인업")
3. 나중에 저장된 라인업에서 "불러오기" 클릭
4. "라인업 저장 및 표시" 클릭 → 오버레이 반영

- "입력 초기화": 모든 입력 필드 비우기

---

## 📋 선수 탭

### 선수 등록
1. 이름, 등번호, 포지션 입력
2. "선수 추가" 클릭
3. 포지션 옵션: 투수, 포수, 1루수~우익수, 내야수, 외야수, 야수

### 선수 관리
- "수정" 버튼: 이름, 번호, 포지션 변경
- "삭제" 버튼: 선수 삭제
- 등록된 선수는 라인업 탭에서 드롭다운으로 선택 가능

---

## 🔒 관리 탭 (관리자 전용)

### 관리자 비밀번호
- 새 비밀번호 입력 후 "변경"
- 4자 이상 필수

### 사용자 비밀번호
- 현재 상태 표시: (설정됨) 또는 (없음)
- "변경": 새 사용자 비밀번호 설정
- "삭제": 사용자 비밀번호 제거 (사용자 로그인 불가)

---

## 경기 당일 운영 순서

1. 컨트롤러 접속 → 로그인
2. 경기설정 탭 → 저장된 경기 "적용" (또는 새로 입력 후 "바로 적용")
3. 라인업 탭 → 저장된 라인업 "불러오기" → "라인업 저장 및 표시"
4. 경기 탭으로 이동
5. "스코어 ON" → 스코어보드 오버레이 표시
6. 경기 진행하며 점수/이닝/카운트/베이스 조작
7. 라인업 보여줄 때: "라인업 ON" / 숨길 때: "라인업 OFF"

### 학년 교체 시
1. 라인업 탭 이동
2. 저장된 라인업에서 해당 학년 "불러오기"
3. "라인업 저장 및 표시"
4. 경기 탭으로 돌아가서 계속 진행

---

## 문제 해결

| 증상 | 해결 |
|------|------|
| 오버레이가 안 보임 | "스코어 ON" 또는 "라인업 ON" 확인 |
| 로그인이 안 됨 | 관리자가 사용자 비밀번호를 설정했는지 확인 |
| 라인업이 비어 보임 | "라인업 저장 및 표시" 다시 클릭 |
| 로고가 안 올라감 | 500KB 이하 이미지만 가능 |
| 선수 목록이 안 뜸 | 선수 탭에서 선수를 먼저 등록 |
